# TraumaShakeContainer Integration with PixiJS v7

## Overview

The `TraumaShakeContainer` is a powerful utility that enables developers to apply shake effects to any display object within a PixiJS v7 application. This guide provides a structured approach to integrating the `TraumaShakeContainer` into your application, complete with event handling for a trigger using the spacebar.

## Audience

This documentation is intended for developers who are familiar with JavaScript and the PixiJS framework, seeking to enhance their applications with dynamic shake effects.

## Prerequisites

Before proceeding, ensure the following dependencies are included in your project:

```bash
npm install pixi.js
```

## Setting Up Your PixiJS Application

Follow these steps to establish a basic PixiJS application that utilizes the `TraumaShakeContainer`.

import { Steps } from '@astrojs/starlight/components';

<Steps>

1. **Create a Basic PixiJS Application**

   Create an HTML file with a canvas element where your PixiJS application will render. Then, in your JavaScript file, initiate the PixiJS application.

   ```html
   <!DOCTYPE html>
     <html lang="en">
     <head>
         <meta charset="UTF-8">
         <meta name="viewport" content="width=device-width, initial-scale=1.0">
         <title>PixiJS TraumaShakeContainer Example</title>
         <script src="app.js" defer></script>
     </head>
     <body>
         <canvas id="pixiCanvas"></canvas>
     </body>
   </html>
   ```

   ```javascript
   import { Application } from "pixi.js";
   import { TraumaShakeContainer } from "./TraumaShakeContainer";

   // Initialize PixiJS application
   const app = new Application({
     width: window.innerWidth,
     height: window.innerHeight,
     backgroundColor: 0x104488,
     view: document.getElementById("pixiCanvas"),
   });
   ```

2. Create the TraumaShakeContainer:

   Instantiate the `TraumaShakeContainer` to create a shakeable container. This container will hold your graphical elements, allowing them to shake when triggered.

   ```javascript
   const shakeContainer = new TraumaShakeContainer();
   app.stage.addChild(shakeContainer);
   ```

3. Add Visual Elements:

   For demonstration, add a simple rectangle to the shake container.

   ```javascript
   const rectangle = new PIXI.Graphics();
   rectangle.beginFill(0xffc0cb);
   rectangle.drawRect(0, 0, 100, 100);
   rectangle.endFill();
   shakeContainer.addChild(rectangle);
   ```

4. Implement Spacebar Event to Trigger Shake Effect

   By listening for the `keydown` event on the window, you can trigger the shake effect when the spacebar is pressed. Add this functionality to your application:

   ```javascript
   window.addEventListener('keydown', (event) => {
     if (event.code === 'Space') {
       shakeContainer.shake();  // Call the shake method to initiate shaking
     }
   });
   ```

</ Steps>

### Full Implementation

Integrating all the steps, your `app.js` file should look like this:

```javascript
import { Application } from "pixi.js";
import { TraumaShakeContainer } from "./TraumaShakeContainer";

const app = new Application({
  width: window.innerWidth,
  height: window.innerHeight,
  backgroundColor: 0x104488,
  view: document.getElementById("pixiCanvas"),
});

const shakeContainer = new TraumaShakeContainer();
app.stage.addChild(shakeContainer);

const rectangle = new PIXI.Graphics();
rectangle.beginFill(0xffc0cb);
rectangle.drawRect(0, 0, 100, 100);
rectangle.endFill();
shakeContainer.addChild(rectangle);

window.addEventListener('keydown', (event) => {
  if (event.code === 'Space') {
    shakeContainer.shake();  // Initiates the shake effect
  }
});
```
